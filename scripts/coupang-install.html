🛒 퐁당샤브 쿠팡 동기화 설치

5초만에 끝나는 셋업. 한 번만 하면 됨.

1단계 — 즐겨찾기 막대 보이기

크롬 위쪽에 즐겨찾기 막대가 안 보이면:

Ctrl+Shift+B 누르기 (맥은 ⌘+Shift+B)

2단계 — 아래 버튼을 즐겨찾기 막대로 끌어다 놓기

🛒 퐁당쿠팡 동기화
↑ 위 버튼을 마우스로 잡고 끌어서 즐겨찾기 막대에 놓기

드래그가 안 되면 (수동)

아래 [복사] → 즐겨찾기 막대 우클릭 → [페이지 추가] → 이름 "🛒 퐁당쿠팡 동기화" + URL 칸에 붙여넣기

3단계 — 쿠팡 주문내역 페이지에서 클릭

① 크롬에서 https://mc.coupang.com/ssr/desktop/order/list 열기 (로그인 상태)

② 즐겨찾기 막대의 🛒 퐁당쿠팡 동기화 클릭

③ 화면에 "동기화 완료 — 새로 N건" 뜨면 성공

그 다음 흐름:
앱 켜기 → 사이드메뉴 → 거래처 → 쿠팡 진입 → "🆕 N건 분류 대기" 노란 배너 → 탭 → 카테고리 확인 후 [✓ 저장]

⚠️ 문제 생기면

F12 → Console 탭에서 빨간 글씨 확인 후 CTO에게 캡처 전달
