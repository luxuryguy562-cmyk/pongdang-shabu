## 🎯 가격 철학 (vision.md 직속)

### vision 1조 (사장님 명언)
> *"손 하나 까딱하기 싫다. 매월 20시간 → 30분."*

### 가격 계산 기준
사장님 정산 시간 절감 = **월 19.5시간** (20시간 - 30분).
시급 환산:
- 사장님 본인 시급 (자영업 기회비용) **최소 1만원** ~ **최대 5만원**
- 시간 절감 가치 = 월 **19.5만원 ~ 97.5만원**
- → **월 9,900원 받아도 사장님 ROI 19.7배** (1만원 기준)
- → 월 1만원 = **압도적 가성비** = 신규 유저 진입 장벽 최소

## 📊 가격 전략

### 🪜 가격 사다리 (사용자 자연 이동)
```
무료 → 베이직(9,900) → 프로(29,900) → 프차(99,000/매장)
   ↑ 영수증 50장 다 씀          ↑ 매장 늘어남
                ↑ AI 인사이트 갖고 싶음
```

### 🎁 첫 14일 무료 체험
- 신규 가입 = 자동 베이직 14일 무료
- vision 7-3 자동 온보딩 = 14일 안에 사장님 "어 이거 편하네" 경험
- 14일 후 자동 베이직 결제 또는 무료 다운그레이드

### 💸 결제 모델
- 월 결제 (기본)
- 연 결제 = **20% 할인** (베이직 9,900 → 95,000원/년 / 프로 29,900 → 287,000원/년)
- 카드·이체·계좌이체 (한국 표준)
- ⚠️ 토스 인앱결제 / 카카오페이 / 네이버페이 연동 (Phase 2+)

## 🎯 양면 시장 (Phase 3) 수익 분리

vision 4번 수익 모델 4단계 중 **2~4번은 SaaS 구독료 외 부수익**:

| 항목 | 수익 형태 | 시점 |
|---|---|---|
| **SaaS 구독료** | 베이직/프로/프차 월 정액 | Phase 1~ |
| **식자재 입점 수수료** | 거래 GMV의 3~5% | Phase 3 |
| **세무·노무 leads** | 건당 50,000~100,000원 | Phase 3 |
| **보험 leads** | 건당 30,000~80,000원 | Phase 3 |
| **대출 leads** ⭐ | 실행 금액의 0.5~1% (캐시노트 모델) | Phase 3 |
| **광고** | CPM/CPC | Phase 4+ |

→ Phase 3 부수익이 SaaS 구독료보다 **3~10배** 클 가능성 (캐시노트 케이스).

## 🚀 1년 매출 5천만원 목표 역산 (vision 4번)

### 시나리오 A: 베이직만
- 5,000만원 ÷ 12개월 ÷ 9,900원 = **약 421명 유료 전환**
- 1,000명 가입 → 421명 베이직 결제 = **전환율 42%** 필요
- ⚠️ 일반 SaaS 무료→유료 전환율 5~10% → **현실적으로 어려움**

### 시나리오 B: 베이직 + 프로 + Phase 3 부수익
- 베이직 200명 × 9,900 × 12 = 약 2,376만원
- 프로 50명 × 29,900 × 12 = 약 1,794만원
- Phase 3 부수익 (대출 1건 + 세무 5건 + 식자재 GMV) ≈ 500만원
- **합계 ≈ 4,670만원 → 1,000만원 더 필요** = 프차 1개 매장 영업

### 시나리오 C: 프차 B2B 1건 + 베이직
- 프차 1개 매장 × 99,000 × 12 = 약 1,188만원/매장
- 5개 매장 = 5,940만원 → **목표 달성**
- 베이직 = 보너스

⚠️ **사장님 검증 필요 2**: 시나리오 A/B/C 중 사장님 직감에 맞는 거?
