## 🔄 외주 vs 정직원 기준

### 외주가 맞는 경우
- 콘텐츠 제작 (인플루언서·디자이너)
- 1회성 작업 (영상·이벤트·박람회)
- 사용자 수 불안정 (Phase 2 초)
- 전문성 필요한데 풀타임 X (세무사·변호사)

### 정직원이 맞는 경우
- 사용자 1,000명+ 안정 (Phase 3 초~)
- 매일 반복 작업 (CS·운영)
- 영업 비밀 다루는 일 (vision 7-1 데이터 정책)
- 입점 업체 관계 관리 (장기 신뢰)
