## 📅 Phase 1 — PoC (지금~2개월 / 2026-05~2026-07)

### 목표
- 우리 가게 1개월 스프레드 수동값 = 앱 자동값 **100% 일치**
- 매출 / 지출 / 영수증 분류 3개 모두

### 작업 항목 (vision 1순위 = 사장님 손 가는 매일 작업 0)

> ⚠️ **2026-05-24 큰 방향 변경 (사장님 명시)**: 옛 "Codef API 대기" 폐기. 진짜 답 = 문자 자동 + 매출 백그라운드.

| 우선 | 작업 | 담당 | 완료 기준 |
|---|---|---|---|
| 🥇 | **안드로이드 네이티브 (Capacitor 래핑) + SMS 자동 읽기** | Claude (coder) | 카드·계좌 SMS 도착 → DB 자동 INSERT |
| 🥇 | **Gmail OAuth 백그라운드** (iOS 보완 + 카드 명세 이메일) | Claude | 가입 시 OAuth 1회 → 명세 자동 인식 |
| 🥇 | **POS API 매출 백그라운드 강화** | Claude | 업솔루션 자동 매일 매출 가져옴 (사장님 마감 입력 ↓) |
| 🥇 | **정산 100% 일치 검증** | 사장님 + Claude | 1개월 스프레드 수동값 = 앱값 |
| 🥈 | 거래처 명세서 OCR 강화 | Claude | 명세서 사진 1장 → DB 자동 입력 (현 #45 #60 #61 추가 강화) |
| 🥈 | 자동 인사이트 (vision 7-4) | Claude | "인건비 35% 업계 평균 +5%" / "공과금 미납" / "단가 변동" |
| 🥉 | 매출/지출/순익 한 화면 강화 | Claude (designer) | 대시보드 "남는 돈" 직관 보강 |
| 🥉 | 7개 docs 매월 정리 | Claude | work_log.md / dev_lessons.md 비대 정리 (헌법 11조) |

### KPI
- 사장님 정산 시간 = 월 30분 (현재 20시간)
- 정산 정확도 100%
- 자동 인사이트 월 5개 이상 자동 생성

### 사장님 작업
- 1개월치 스프레드 → 앱 자동값 비교 (검증)
- 거래처 명세서 5~10장 보내주기 (OCR 학습용)
- vision/persona/pricing 검증 답변
- **Capacitor 안드 빌드 후 사장님 폰에 설치 + SMS 권한 1회 허용**
- **Gmail OAuth 1회 허용 (Google 계정 카드사 명세 이메일 인식용)**
