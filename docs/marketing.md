## 📊 1,000명 모집 역산

### 깔때기 (Funnel) 가정
| 단계 | 전환율 | 필요 인원 |
|---|---|---|
| 광고·콘텐츠 노출 | — | **50,000명** |
| 랜딩 페이지 방문 | 4% | 2,000명 |
| 회원가입 (무료) | 50% | 1,000명 |
| 14일 체험 + 유료 전환 | 30% | 300명 (베이직+프로) |

→ 노출 5만 → 가입 1,000 → 유료 300명 → 매출 ≈ 약 2,000만~3,500만원
→ 5천만 목표 도달하려면 = 프차 B2B 1~2건 추가 (pricing.md 시나리오 C)

## 🚀 채널별 전략 (우선순위)

### 🥇 1순위. SNS 자체 콘텐츠 (가성비 1위)
- **인스타그램** — 식당 사장님 일상·정산 비포애프터 릴스
  - "월 20시간 정산 → 30분 (실제 사장님 사용 영상)"
  - 메인 페르소나가 가장 많이 머무는 채널
- **유튜브 숏츠** — 1분 자동화 시연
- **블로그** — SEO ("식당 자동 정산", "캐시노트 대안" 검색 노출)

**예산**: 콘텐츠 제작비 = 사장님 직접 (퐁당샤브 실제 사용 = PoC 영상)
**KPI**: 월 팔로워 +500 / 영상 평균 조회 5,000

### 🥈 2순위. 식당 사장님 커뮤니티 침투
- **네이버 카페** — "음식점 사장 모임", "프랜차이즈 정보" 등
- **카카오 오픈채팅** — 식당 운영 채팅방
- **인스타 식당 해시태그** — #식당사장 #자영업 #식당운영

**전략**: 직접 홍보 X → "사장님이 직접 만든 어플" 진정성 콘텐츠
**KPI**: 월 50명 유입

### 🥉 3순위. 베타 사용자 입소문 (Phase 1 → Phase 2 다리)
- 베타 10명 → 만족 시 **사장님 1명 = 식당 사장님 3~5명 추천**
- **추천 보너스**: 추천인 + 피추천인 모두 1개월 무료 (베이직)
- **vision 7-4 자동 인사이트** = 입소문 무기 ("이번 달 인건비 비율 높음" 직접 보여주면 다른 사장님도 갖고 싶어함)

### 🎯 4순위. B2B 영업 (Phase 2 후반)
- 프랜차이즈 본사 직접 영업 (퐁당샤브 본사 등 사장님 네트워크 활용)
- 외식업 협회·창업 박람회 부스
- **KPI**: 분기 1~2건 B2B 계약 = 5천만 매출 달성 핵심

### 5순위. 유료 광고 (보조)
- 페이스북·인스타 광고 (1차 검증 후만)
- 구글 검색 광고 ("식당 정산 어플" 등)
- **예산**: 베타 검증 끝나고 시작 (예산 낭비 방지)

## 🎁 캠페인 아이디어

### 1. "식당 사장님 정산 챌린지"
- 캐시노트 vs 우리 어플 vs 스프레드시트 = 같은 1개월 데이터로 정산 시간 측정
- 우리가 압도적이면 = 영상 그대로 광고

### 2. "비포애프터 챌린지"
- 사용자 본인 정산 시간 비포(스프레드 20시간) / 애프터(앱 30분) 자랑 영상
- #퐁당정산챌린지 (가칭)
- 베스트 영상 1년 무료

### 3. "친구 데려오면 1개월 무료" (P2P)
- 추천 1명 = 추천인·피추천인 둘 다 1개월 무료
- 추천 5명 = 평생 베이직 무료

### 4. "초기 100명 평생 50% 할인"
- Phase 1 → 2 다리 시기 1차 사용자에게 평생 할인
- 충성 사용자 확보 + 입소문
