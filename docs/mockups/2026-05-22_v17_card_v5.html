🎨 디자인 의도

  • 순수익 = 적당히 강조 (20px 초록색). 카드 중앙 + 위아래 점선으로 분리 → 시선 자연 흡수. 졸라 X.
  • 좌·우 일관성: 좌측 = 실제, 우측 = 예상. 예상은 회색톤 + "예상" 작은 라벨 + 75% 투명도 → "이건 미래"
  • 매출 ▲12% vs 순수익: 매출 옆에 % 작게, 순수익은 라벨만 + 금액 자체로 강조 (시선 분산 X)
  • 간트 두 줄: 위 = 지출 분포(카테고리별), 아래 = 매출 대비 순수익/지출 비율. 한눈에 "어디 얼마 썼고, 얼마 남았나"
  • 상위 3개: 회색 박스 안 깔끔 행 — 카드 안 작은 위계로 통합. 색깔 점 + 이름 + % + 전월대비
  • 상세 보기: 누르면 전체 카테고리 풀 표 (이름·금액·%·전월대비 4컬럼). 합계 행 강조선.
  • 전월 대비 코멘트: 카드 맨 아래 (현재 그대로). 활기·차분·쉬어가는 등 사장님 단어로 자동 코멘트.
