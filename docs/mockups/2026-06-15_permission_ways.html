권한, 어떻게 만들까 — 두 방식

사장님이 실제로 누르는 화면 차이
방식 1 내가 하나씩 켜고 끔
직급 고른 뒤, 화면마다 스위치를 직접 켜고 끕니다. 제일 자유롭지만 손이 갑니다.
📋 점장 — 볼 수 있는 화면
🏠 홈
⏰ 근태관리
🍳 개시·마감
💳 지출관리
⋯ 더보기
→ 사장님이 직급마다 스위치를 일일이 켭니다 (자유 ↑ / 손 ↑)
방식 2 직급만 고르면 자동
직원의 직급만 고르면, 그 직급에 맞는 화면이 자동으로 켜집니다. 사장님은 스위치 안 만집니다.
👤 이송은 직원 — 직급 선택
팀장 ▾
팀장을 고르면 자동으로 이 화면들이 켜져요:
🏠 홈⏰ 근태관리
(개시·마감 / 지출 / 더보기는 자동으로 꺼짐)
→ 사장님은 "팀장" 한 번만 고르면 끝 (빠름 / 세밀 조정은 나중)
방식 1 = 내가 스위치 다 켬 (자유롭지만 번거로움)
방식 2 = 직급만 고르면 자동 (간단)

"방식 1" 또는 "방식 2"로 답해주세요
