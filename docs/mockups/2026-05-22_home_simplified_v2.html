💡 핵심 결정 사항 (designer 자동 적용)

  • "오늘 매출 과한감" 반영 — 30px → 17px 절제 + 회색톤. 매출/지출/순익 3분할로 시각 부담 분산.
  • 같은 카드 양식 — 일/주/월 모두 동일 (매출/지출/순익 3분할 + 정확 금액 한 줄). 통일감.
  • 캘린더가 진입로 — 셀 탭하면 카드만 변동. 새 화면 X. dev_lessons #51 "데이터=진입로" 정확.
  • 오늘 자동 선택 — 진입 시 캘린더에 오늘 강조 + 카드 = 오늘 정보. 사장님 디폴트 1순위 정보 즉시 노출.
  • 주별 카테고리 비교 — 주 모드일 때만 노출. 월 모드에서는 사라짐 (시각 노이즈 ↓).
  • 기타매출 — 사이드메뉴 정리. 매장별 조건부 + 매일 안 보는 정보.
